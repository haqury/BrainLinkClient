# BrainLink Client - Обзор проекта

## 📋 Что это?

**BrainLink Client** - Python приложение для управления компьютером с помощью мысли, используя EEG (электроэнцефалограмма) данные от устройства BrainLink.

Это полноценный порт C# приложения BrainLinkConnect на Python с современной архитектурой и расширенными возможностями.

## ✨ Ключевые возможности

### 🧠 Управление мыслью
- Двигайте курсор мыши силой мысли
- Обучаемая система распознавания паттернов
- Многоуровневая фильтрация для точности

### 🎯 Гироскоп
- Управление движениями головы
- Калибровка положения
- Определение направлений (вверх/вниз/влево/вправо)

### 📊 Визуализация данных
- Отображение EEG волн в реальном времени
- Мониторинг Attention и Meditation
- Просмотр расширенных данных (пульс, температура)

### 💾 Обучение и сохранение
- Запись паттернов мозговой активности
- Сохранение/загрузка истории
- Конфигурируемая чувствительность

## 🚀 Быстрый старт

### 1. Установка
```bash
python -m venv venv
source venv/bin/activate  # Linux/macOS
venv\Scripts\activate     # Windows
pip install -r requirements.txt
```

### 2. Запуск
```bash
python main.py
```

### 3. Использование
1. Подключите BrainLink устройство
2. Обучите паттерны (10-15 минут)
3. Включите автоматическое управление
4. Думайте - курсор двигается!
