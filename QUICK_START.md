# Быстрый старт BrainLink Client

## Установка за 5 минут

### Шаг 1: Подготовка
```bash
# Windows
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt

# Linux/macOS
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

### Шаг 2: Запуск
```bash
python main.py
```

Или используйте готовые скрипты:
- Windows: двойной клик на `run.bat`
- Linux/macOS: `./run.sh`

## Первое использование

### 1. Подключение устройства

```
┌─────────────────────────────┐
│  Главное окно               │
├─────────────────────────────┤
│  [Start & Connect] ← Клик   │
└─────────────────────────────┘
         ↓
┌─────────────────────────────┐
│  Окно подключения           │
├─────────────────────────────┤
│  □ BrainLink-XXXX           │
│  □ Device-YYYY              │
├─────────────────────────────┤
│  [Выбрать] [Connect]        │
└─────────────────────────────┘
```

### 2. Обучение паттернам (3 минуты)

**Для каждого направления (Left, Right, Up, Down):**

1. Выберите направление (например, ☑ Move Left)
2. Нажмите и держите клавишу ← (стрелка влево)
3. Сконцентрируйтесь на мысли "влево" 10 секунд
4. Отпустите клавишу
5. Повторите для других направлений

**Совет:** Запишите 20-30 образцов для каждого направления

### 3. Настройка чувствительности

```
[Config Fault] → Установите допуски:
┌──────────────────────────────┐
│ Attention:    10             │
│ Meditation:   10             │
│ Delta:        50000          │
│ Theta:        50000          │
│ Low Alpha:    20000          │
│ High Alpha:   20000          │
│ Low Beta:     15000          │
│ High Beta:    15000          │
│ Low Gamma:    10000          │
│ High Gamma:   10000          │
├──────────────────────────────┤
│ Multi Count:  3              │
└──────────────────────────────┘
```

**Правило:** 
- Большие значения = менее точно, но срабатывает чаще
- Меньшие значения = точнее, но может не срабатывать

### 4. Включение управления

```
☑ Auto Use (Direct Control)    ← Включить
☑ Use Key Control              ← Включить
```

Теперь думайте о направлении - курсор должен двигаться!

## Базовые команды

### Сохранение прогресса
```

[Save to File]  ← Сохранить обучение
```

```

[Load from File]  ← Загрузить обучение
```

### Очистка истории
```
[Clear]  ← Удалить все записи
```

## Типичные проблемы

### Курсор не двигается
✓ Проверьте: записана ли история обучения (Records > 0)
✓ Проверьте: включены ли чекбоксы Auto Use и Use Key Control
✓ Попробуйте: увеличить допуски в Config Fault

### Устройство не подключается
✓ Bluetooth включен?
✓ BrainLink заряжен?
✓ Попробуйте: перезапустить приложение

### Паттерны путаются
✓ Запишите больше образцов (50+ на направление)
✓ Уменьшите допуски в Config
✓ Увеличьте Multi Count до 5

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| ←       | Обучение "Move Left" |
| →       | Обучение "Move Right" |
| ↑       | Обучение "Move Up" |
| ↓       | Обучение "Move Down" |

## Советы по эффективному использованию

1. **Обучайтесь в спокойном состоянии**
   - Расслабьтесь перед записью
   - Избегайте отвлекающих факторов

2. **Записывайте разнообразные образцы**
   - В разное время дня
   - В разных состояниях концентрации

3. **Регулярно сохраняйте историю**
   - Создавайте резервные копии
   - Экспериментируйте с разными профилями

4. **Калибруйте гироскоп каждый раз**
   - После надевания устройства
   - При изменении положения
