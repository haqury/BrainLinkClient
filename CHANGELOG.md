# Changelog

Все значимые изменения в проекте BrainLink Client будут документированы в этом файле.

## [1.0.0] - 2026-01-19

### Добавлено
- Первоначальный релиз Python версии BrainLinkConnect
- Структура проекта с модулями models, services, ui
- Главное окно приложения (MainWindow)
- Окно подключения к Bluetooth устройствам (ConnectForm)
- Окно отображения EEG данных (EEGDataForm)
- Окно гироскопа (GyroForm)
- Окно конфигурации (ConfigForm)
- Сервис управления историей (HistoryService)
- Сервис отслеживания движений головы (HeadTracker)
- Сервис управления мышью (MouseService)
- Сервис Bluetooth подключения (BluetoothService)
- Модели данных: EEG, Gyro, Config, System
- Система распознавания паттернов EEG
- Многоуровневая система фильтрации событий
- Сохранение/загрузка истории в JSON
- Сохранение/загрузка конфигурации в JSON
- Глобальный перехват клавиатуры для обучения
- Режим автоматического управления курсором
- Калибровка гироскопа
- Скрипты запуска для Windows (run.bat) и Linux/macOS (run.sh)
- Подробная документация (README.md, INSTALL.md)
- Пример конфигурации (example_config.json)

### Технические детали
- Python 3.8+
- PyQt5 для GUI
- bleak для Bluetooth
- pyautogui для управления мышью
- pynput для перехвата клавиатуры
- numpy для математических операций

### Отличия от C# версии
- Использование asyncio для асинхронной работы
- Type hints для лучшей читаемости кода
- Кроссплатформенность (Windows, Linux, macOS)
- Упрощенная установка через pip
- Модульная архитектура с четким разделением слоев

### Известные ограничения
- Требуется реализация протокола BrainLink SDK
- Bluetooth подключение не протестировано с реальным устройством
- HRV форма не реализована
- State/Raw data форма не реализована
- UDP сервер не реализован
